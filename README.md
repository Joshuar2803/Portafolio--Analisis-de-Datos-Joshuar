# Portafolio: Análisis y Ciencia de Datos - Joshuar
Bienvenido a mi portafolio profesional. Este espacio documenta mi evolución y aprendizaje en el mundo de los datos. Mi objetivo es desarrollar soluciones analíticas robustas, transformar datos crudos en insights estratégicos y, eventualmente, construir sistemas predictivos y arquitecturas de ingeniería de datos escalables.

# 🚀 Sobre mí
Soy un apasionado de los datos en constante formación. Me especializo en la limpieza, exploración, análisis y visualización de información para la toma de decisiones empresariales. Actualmente, estoy fortaleciendo mis habilidades en el stack tecnológico de Python (Pandas, Seaborn, Scikit-learn) y SQL, Power Bi, Estadistica aplicada, Machine Learning con el fin de aportar valor tangible a cualquier organización.

# 📂 Proyectos Destacados
[Día 06]: Análisis de Rotación de Personal (HR Analytics)
Este proyecto consiste en un análisis integral de rotación laboral, donde se evaluaron factores como salario, carga de trabajo y satisfacción de los empleados.

Tecnologías: Python (Pandas, Matplotlib/Seaborn), SQL.

# Logros:

Identificación de brechas salariales críticas y su relación directa con la rotación.

Comparativa técnica entre procesamiento en Pandas vs. consultas SQL.

Entrega de un Resumen Ejecutivo orientado a la toma de decisiones estratégicas.

# 🛠 Habilidades Técnicas
Análisis y Manipulación: Pandas, NumPy, SQL (Joins, Aggregations).

Visualización: Matplotlib, Seaborn, Power BI, Looker Studio, Tableu, Qt Designer(UI para aplicaciones de datos).

Fundamentos: Limpieza de datos, estadística descriptiva, análisis exploratorio (EDA).

# 📈 Mi Ruta de Aprendizaje
